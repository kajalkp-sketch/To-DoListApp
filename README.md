A simple and interactive Todo List Web Application built with HTML, CSS, and JavaScript. This app helps users organize tasks, mark them as complete, and manage their daily productivity in a clean, responsive interface.

🚀 Features:

➕ Add new tasks easily
✅ Mark tasks as completed
🗑️ Delete tasks when no longer needed
💾 Local storage support (tasks persist even after refreshing the page)
📱 Responsive design for desktop and mobile

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Storage: Browser LocalStorage
